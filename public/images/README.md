# Images Directory

This directory contains static images used throughout the application.

## Cockpit Transition

Place your cockpit image here:

**File:** `cockpit-image.jpg` (or `.png`)

**Path:** `/public/images/cockpit-image.jpg`

**Usage:** Referenced in cockpit transition components as `/images/cockpit-image.jpg`

---

## Image Recommendations

- **Format:** JPG (for smaller file size) or PNG (for transparency if windshield is cut out)
- **Optimization:** Consider compressing the image to reduce load time
- **Dimensions:** Match viewport size or larger (e.g., 1920x1080 or 2560x1440)
